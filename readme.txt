== Installation ==

1. Install easily with the WordPress plugin control panel or manually download the plugin and upload the folder to the /wp-content/plugins/ directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Create or edit a menu by clicking the "Easy Calendar" admin menu link
4. To display on a page or post use shortcode [tortuga-recurring-calendar]
